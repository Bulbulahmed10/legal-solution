The main different between flexbox & grid is
Flexbox is primarily used for one-dimensional layout, meaning that it is best suited for arranging items in a single row or column. It is most commonly used for creating responsive navigation bars, grid layouts with flexible items, and aligning items within a container.

CSS Grid, on the other hand, is best suited for two-dimensional layout, meaning that it can handle both rows and columns. It is most commonly used for creating grid-based layouts, such as photo galleries, and can also be used for creating responsive designs.

Media Queries are used for responsive design
allowing for web pages to be adjusted to different screen sizes, resolutions and orientations, making the website more user-friendly and accessible.

The CSS box model describes how the dimensions and spacing
of elements in a webpage are calculated. It defines a rectangular area around each HTML element, called a box, that consists of four main parts: content, padding, border and margin, and the total width and height of an element is the sum of these four parts.

let's say we have an element with a width of 100px, padding of 10px, border of 2px, and margin of 20px. The total width of the element would be: 100px (content) + 10px (padding) + 2px (border) + 20px (margin) = 132px.

It's important to note that the default box-sizing value is content-box, which means that the total width and height of an element is calculated as described above. However, you can change the box-sizing property to border-box, which means that the padding and border are included in the width and height of the element. This can make it easier to control the size and layout of elements on a webpage.

Semantic tags are HTML tags that provide meaning to the
structure of the content on a webpage, describing the type of content within an HTML element, making it easier for browsers, search engines, and screen readers to understand the purpose of the content. Using semantic tags can make your code more readable. Examples of common semantic tags include:
<header>, <nav>, <main>, <article>, <section>, <aside>, and <footer>
